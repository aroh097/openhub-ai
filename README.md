# Open Hub AI (Ready Project)

## Steps to Run

1. Install Node.js
2. Open terminal in this folder
3. Run:
   npm install
   node server.js

4. Open index.html in browser

## IMPORTANT
- Add your OpenAI API key in environment variable:
  OPENAI_API_KEY=your_key_here

## Deploy
- Backend → Render
- Frontend → Vercel
